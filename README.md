# Explosive Salad
A multiplayer prototype made with Godot Engine 4.1.

## Features
- Animated character
- Quake-like controller
- PVP
- Sound effects
- Explosions

## Plans
- Gibs
- Scoreboard
- More maps
- Mod support
- Gamepad support
- More gameplay features

## Controls

| Key | Command |
| --- | --- |
| <kbd>W</kbd> <kbd>A</kbd> <kbd>S</kbd> <kbd>D</kbd> | Movement |
| <kbd>Spacebar</kbd> | Jump |
| <kbd>Left mouse button</kbd> | Shoot |
